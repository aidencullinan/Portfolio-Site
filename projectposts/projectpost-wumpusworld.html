Wumpus World Agent-Based Decision-Making System

Expert System for the Wumpus World

Overview

For our first project in Foundations of AI, we developed an intelligent agent that navigates the classic Wumpus World environment using expert system principles. Our implementation leverages a knowledge base (KB), environmental parsing, and logical reasoning to infer safe and dangerous zones in a 4x4 grid, aiming to retrieve gold while avoiding deadly pits and the lurking Wumpus.

Project Goals

  • Implement a rule-based expert system to reason about sensory data (breeze, stench, glitter).
  • Simulate agent decision-making between safe, uncertain, and risky moves.
  • Design a knowledge base system (kb.txt) to log movement, inference rules, and game outcomes.
  • Allow users to query the agent’s knowledge of specific chambers in the grid.
  • Track logical deductions and probabilities for pit/Wumpus presence based on prior observations.

Key Features

Environment Setup

  • Parses maps from testworld.txt, testworld2.txt, or testworld3.txt.
  • Dynamically places Wumpus, gold, and pits in a 4x4 grid using input files.

Knowledge-Based Agent

Sensors: Detects breeze, stench, glitter, gold, pit, and Wumpus.

Logical Inference:

  • Breeze → Pit nearby
  • Stench → Wumpus nearby
  • No breeze/stench → Safe adjacent rooms

Decision Models:

  • startAgent_random() – random movement (baseline)
  • startAgent_expert_v1() – logic-driven movement based on KB

Logging and Analysis

  • Tracks agent's movements, deaths, and wins in the KB file.
  • Logs deductive reasoning for each move.
  • Exports current world state and agent knowledge for debugging or review.

Interactivity

  • Users can test the agent’s memory by querying chamber safety.
  • Responds with status: SAFE, Breeze detected, Stench detected, or UNKNOWN.

Sample Agent Behavior

  • Begins at (1,1) (safe zone).
  • Checks for breeze or stench.

  • If no safe moves, reroutes to a known safe cell or makes a risky decision.

Conclusion

This project deepened our understanding of how expert systems simulate intelligence using logic, inference, and environmental sensing. By representing knowledge as rules and facts, we designed a flexible and scalable agent that can safely traverse the dangers of the Wumpus World.
